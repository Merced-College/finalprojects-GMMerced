# cpsc39-finalProjects

Money Tracker - CPSC-39 Final Project

Name: Goutham Mahesh  
Class: CPSC-39 (Professor Kanemoto)  

This program is a money tracker that allows a user to log what they earn and what they spend. It helps anyone wishing to save money or observe their spending habits. The user can easily add or remove transactions. The program can sort the money by date or by the amount. It will calculate and show a summary of the money earned, money spent, and the final balance. It also saves the user's data into a file and reads it when the program is run again.
